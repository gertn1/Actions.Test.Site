# Actions.Test.Site API

## Descrição

Actions.Test.Site é uma API desenvolvida em C# para gerenciamento de usuários, autenticação com JWT e controle baseado em papéis. Ela permite criar, editar, listar e excluir usuários, além de aplicar restrições de acesso a rotas de acordo com os papéis atribuídos.

## Tecnologias Utilizadas
- .NET 8
- Entity Framework Core
- FluentValidation
- JWT Authentication
- BCrypt.Net para criptografia de senhas

## Pré-requisitos

- [.NET 6 SDK](https://dotnet.microsoft.com/download/dotnet/6.0)
- [SQL Server](https://www.microsoft.com/en-us/sql-server/sql-server-downloads) ou outro banco de dados compatível.
- Git
